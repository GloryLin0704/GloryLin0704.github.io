The Last For 2018

发表于 2018-12-31

前言

2018 年，很累的一年。所幸已是最后一天。

曾经梦想仗剑走天涯 现实柴米油盐酱醋茶

学业

上半年，有惊无险度过，没想到大二还能混个奖学金，是有点小意外。

然而，这学期开始上专业课了，不过都好无聊，软件开发工具、软件项目管理，还有下学期的软件测试，毫无兴趣，幸好这些课都是交报告。

说到感兴趣的课呀，那我很快就想到了数据库老师和蔼的面容，哈哈。这一年里最喜欢的课，就是数据库，遇到你，花光四年的课程。

悄悄地，即将走完大三上了，时间如流水，不经意间就快毕业了。回想起刚入学，和现在的自己对比，似乎，并没有什么不同。

果然很是讨厌读书，幸好并没有考研的打算

编程

这一年，做了很多乱七八糟的事情。

3 月份机缘巧合，接触过一下 React，因为要用 飞冰 做点东西，那时候的飞冰只有 React 框架，所以一脸懵逼去弄了弄，心里暗想，为什么要那么沙雕把 DOM 也写在一起，变量修改还要用 setState。后来才知道，原来这是 JSX ，原来我才是沙雕。

之后，再因为机缘巧合，接触了 SEO，果然是个大杀器，愣是研究了很久都不懂。问了很多师兄，查了很多资料，大概大概懂了点，百度蜘蛛收录的时候，并不会执行脚本，所以用 vue 写出来的 SEO 效果非常差。于是乎，在 webpack 中配了 prerender-spa-plugin，去预渲染首页，稍微缓解了一下。好吧，除了服务端渲染，我想不到其他办法去解决了。

后面，用 koa2 去写了许愿墙的后台，发现和 express 写法上是有一点点区别，koa2 中基本都使用了 ES6 的语法，不过本质上我觉得还是一样，洋葱模型。

然后，就是终于学会了 接入公众号网页开发，2 月份的时候，研究了很久都没看懂微信如何授权，如何获取用户信息，直至 10 月份，公众号网页开发这块算是 ok 了。当时还想着写篇博客记录一下怎样接入，现在才忽然想起来忘了这事。。。

玄不改命，氪不救非

做了很多东西之后，回头看看，发现这一年，编程水平并没有很显著提升，只是搬砖的速度越来越快，今年得到提升的大概是解决问题的能力吧，总是能遇到很多灵异问题，也是服气。

从 11 月开始，除了必要的项目跟进，基本没有再继续写代码了，回头看看 js 的原理，过上中年油腻生活，时常上一些论坛看看文章，开拓一下知识面，或者是去实现一些常用的东西。感觉有很多东西不会，但又不知道从哪里开始。

  • 函数式编程，没掌握
  • SSR，了解一点点 Nuxt.js ，没实践过
  • Koa2，尚且入门
  • 计算机网络，七七八八

开始关注实习，身边有些优秀的家伙，已经找到了寒假实习，似乎自己还没有这样的能力。寒假留下来继续补补。。

表面看起来稳如老狗，实则慌得一批

生活

这一年，并没有多么难忘，一整年都很平淡，恍惚间，我还以为是 2017 年，2017 的记忆比 2018 的更深。今年得到了很多，也失去了很多。事情并没有按预期那样发展，所以最后带来了很多麻烦。

这一年，拿了奖学金；做了很多项目，政府级、企业级；也参加过比赛拿了第一名；至少可以有一份看得去的简历。

但是太累，背负了太多东西，如果可以，我也希望出去玩，就算不是去约会，也可以去看看外面。说到底还是责任感在作祟吧，责任在肩上，不完成我会感到很愧疚。我很想放下这个担子，但到现在还没有能找到合适的人，说来真是可笑。

对不起，我就是个废物。

后面两个月，来到了创客，小日子过得还挺滋润的，宿舍 -> 饭堂 -> 创客，每天如此，日复一日。

不管怎样，我也在慢慢成长，心境变化挺大的，毕竟也是大了一岁。

跌宕起伏开了个头，平平淡淡收了个尾

情感

Null

路，荆棘而漫长，愿我们一如既往地坚强

最后

去年留下的问题，现在可以回答了。

今年，果然是很重要、很忙的一年


留给明年的问题，得到自己想要的东西了吗?

就这样了吧，愿我们被世界温柔以待！Hello 2019！


ps:郁设画了两年的画
