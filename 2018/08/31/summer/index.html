抓住八月的小尾巴

发表于 2018-08-31

前言

2018-08-31 09:58:41 拖了一个月的总结，在八月的最后一天，还上了。

学业

原本是有要好好学习的决心，绝不能到期末再去补一波，emmm，果然理想太丰满，还是到了期末前几天才开始好好看书，一边抱佛脚，一边吃鸡。。。总滴来说，大二的绩点还OK，说不定踩了一波狗屎运，还能拿个三等奖学金，嘿嘿。

异想天开做做梦也不是不行，嘿嘿。

编程

5月完成了工作室招新，不知不觉就成了师兄、开发的主力。回头看看，好像并没有能拿得出手的技术去教师弟喔，都是在啃阿莲留下来的精神粮食。

emmm，个人技术方面的话，最近这两个月，用 Vue 用得很多，虽然还不是很溜，但也还OK吧，相反 Node 倒学得很少，零零散散的，基本忘了怎样写。。还是在慢慢蠕动向前吧……

最近的话，在学 Koa ，一来是为了再学学 Node , 二来吧，下学期的许愿墙就准备用 Koa 写后端，不过主要还是微信的这坑难踩。

还是在思考，究竟什么是前端。

然后就是，这学期，前前后后，做了四五个项目，一个是用 React 的，其他都是 Vue ， 之所以用 React ，并非我本愿，完全是因为甲方的需求，只能边查边写的那种，是真的难受，出问题都不知道在哪。。。

很慌，并没有什么特长，开始担心找实习。

生活

工作室搬到了蒲公英，路程有点远，夏天有点热，特别是空调坏了的那周，原来生活还可以这般燥热。

参加了 app大赛 ，拿了校赛第一名，联赛第五名，虽然比赛很有水分，但拿到了综测加分还是挺OK的，嘿嘿。

大四的师兄师姐毕业，emmm，很帅很漂亮

上一学期，除了做做编程搬运工，还不知不觉往产品的方向走了一段路，唉，，，完全是被生活所逼喔。时不时就要去跑跑业务，是真滴狗。

培训师弟，似乎并没有很长时间在工作室，也没教到什么，对不起咯。

emmm，出游，从化温泉别墅两日游，不得不说，比去年舒服多啦，海星海星，嘿嘿嘿嘿……

然后是做了助班，在带一群小小师弟师妹。

阳光越是强烈，阴暗就越是深邃。（白条告诉我的……

情感

对于朋友圈的各种狗粮。我啃，我啃，我努力地啃

最后

太久没写 markdown 了，也一整个学期没更新博客了，一来是自己太懒，二来还是自己太赖，又快到年底了，去年写的年度总结里，希望今年是很重要、很忙的一年，emmm，已经是很忙了，但似乎并没有很重要。

加油咯。
