The Last For 2017

发表于 2017-12-31

前言

果然，又到了2017年的最后一天，即将过去的这一年发生了许许多多的事

学业

今年来，不对，准确的说应该是上大学以来，并没有怎样好好的学习，都是到了要期末考前一个星期才开始看书，好像整个大学就要这样平平庸庸的度过了，这个学期嘛，学习的时间大概就只有一个星期，就因为上个星期要考试才去学了一周。

唯一值得高兴的就是这学期水到了奖学金，一种大学没有遗憾了的感觉油然而生

开考一支笔，做题靠运气
考的不是试，是突击能力

写到文章的后面，突然想起了一件很重要的事，就是，，，四级第一次考试 423 ，是的，没过，这学期比别人多上了30+节的大英课

期待一些有趣的科目……

编程

真正开始编程，应该是从7月份开始的吧，前面那段时间，，，充其量就是一个傻逼在敲键盘而已

年初的时候，想着学习做做后台，装装逼，就学了一下传得很神的 Java ,后来进了工作室，由于技术栈的不同，又跑去学了 Node.js ，从这里开始，才算真正步入编程。那段时间阿楠还在，能够一群志趣相同的人一起学习是真的好。再到后来，发现 Node.js 并不能让我在将来的面试中脱颖而出。为此，在深思熟虑后，又跑去学前端，有了后端的基础，对前端发送的 http 请求这部分是很好理解，到现在为止，已经学了两个月，也算入门了，其实我的想法是前后端结合，做一名 全栈工程师 

路漫漫其修远兮 吾将上下而求索

这半年里，写了很多乱七八糟的代码，并没有拿得出的东西，就不枚举了。比较大的收获嘛，应该是懂了点 Promise ，但对异步并没有了解很好，也逐渐开始适应 ES6 的写法，最近在学前端的两个月里对 JS 的理解也比之前深了。

对了，突然想到了一点很重要的东西，我对套模板是比较反感的，每个网站，我觉得都会有一些属于它自己的东西，而不是通过一个模板，反复改，迎合他人，我是比较喜欢从 0到1 的编程，而不是从 1到1 。

前段时间，我曾对当时的处境有些抱怨，为什么我都那么努力在学前端，大家还是没能把观念转变，没有意识到我的变化。那段时间，迫不及待想做点什么东西证明自己，后来一起做了小程序的项目，发现了自己还有很多不足的地方，的确，是我的心太浮躁了，应该一步步来，对学习前端，我只是在起跑的时候绑了一下鞋带，别人多跑了几十米，我也能追得上，我还有后端的优势。

慢慢来，打好基础，一切都能行

生活

离开了校团委，离开了一群为写一篇推文一起通宵的伙伴，不管怎么说，都是很舍不得，有时候路过教六的时候，都会不经意看上去，看看 208 ，看看 306 ，这里是我大一的所有回忆，和这群人可是比室友还铁呀。印象中是紫林师兄这样说的，对于离开了的人都是 已故 的，已经离开，但还是一见如故 。的确是这样呀，看到他们，仿佛一切都和以前一样，抱怨推文没写完，抱怨亚姐又在群上了发了一节课的语音批评，抱怨又要通宵整理青年之声的数据，最喜欢就是喝星哥的鸡汤，坐巫老师开的车，看啊p的绿帽表情，冷嘉豪的场，还有，和大家在一起。

是呀，虽然已经离开，但还是一见如故

离开了校团委，来到了 NW ，120 ，又是一群新的伙伴，一起学编程，一起开车开黑。对了，群里让我感觉最不可能脱单两位，都相继脱单了，我感觉到了这个世界深深的恶意。想了好一阵，虽然已经在一起半年了，但好像并没有什么东西写，可能有些东西是到了离别才会一涌而出。对了，是阿楠，很不幸，阿楠已经离开了我们，呸，是离开了中国，去了加拿大，从今以后，我在地球这边，你在，那边 。知道你要去加拿大的时候还是在7月，那时候想着时间还有很多，没想到，现在的你已经在加拿大了，时间果然不会对谁宽容过。

人生的每段旅程都是值得回忆和珍藏的

情感

undefined

最后

有点嗨，毕竟昨晚写数据库到3点，准备睡觉，我经历了一件很神奇的事情，一个室友打呼噜，三种呼声切换自如，我还故意大叫了几次，都没能吵醒他，还越来越嗨，可能是到4点多我才睡着的吧。。醒的时候已经是今天11点了。。。

如果是100分，今年我拿了80，明年，将会是很重要的一年，可能在下一年的总结里，我会这样说

  • 今年，果然是很重要、很忙的一年

就这样了吧，愿我们被世界温柔以待！Hello 2018！
